HOW TO CONNECT

METHOD 1 — USB + ADB (Best for Android 16)

1. Settings → About Phone → tap Build Number 7×

2. Settings → Developer Options → USB Debugging ON

3. Plug USB cable → Linux / Chromebook

4. Mint Scan → USB Sync → tap ↺ RESCAN

5. Tap ALLOW on phone popup

6. Tap 🚀 OPEN COMPANION ON PHONE

7. Phone opens this page at http://localhost:8765

METHOD 2 — Wi-Fi (No cable)

1. Mint Scan → Wireless tab → START SERVER

2. Note the IP shown (e.g. 192.168.1.100:7777)

3. Phone browser → type that IP:Port

4. Use Sync tab → enter IP → TEST CONNECTION
